# Content Publish Pipeline

## Шаги

1. **Создание контента (EN)**
   - Определить тип: страница / блог-пост / описание объекта / гид по району
   - Написать на английском (базовый язык)
   - Соблюдать Tone of Voice: тёплый, уверенный, storytelling

2. **SEO-оптимизация**
   - Определить целевые ключевые слова (по каждому языку)
   - Включить ключевые слова в title, h1, первый абзац
   - Написать meta description (150-160 символов)
   - Добавить internal links на релевантные страницы

3. **Перевод на 5 языков**
   - Запустить translate-content skill
   - Культурная адаптация (не просто перевод!)
   - Проверить SEO ключевые слова на каждом языке

4. **Публикация в CMS**
   - Загрузить все языковые версии в Sanity
   - Привязать изображения (оптимизированные через Cloudinary)
   - Настроить slug для каждого языка

5. **Проверка**
   - Запустить seo-audit skill
   - Проверить отображение на всех 5 языках
   - Проверить мобильную версию

6. **Деплой**
   - Запустить deploy-preview skill
   - Убедиться в корректности preview
   - Мерж в production
