# Аудит дизайна страницы

## Когда использовать
Когда нужно проверить качество дизайна и производительности конкретной страницы сайта TimeHome Realty.

## Шаги

1. **Откройте страницу в браузере** через browser_subagent и сделайте полноэкранный скриншот.

2. **Проверьте визуальные критерии:**
   - [ ] Hero-секция занимает 100% viewport
   - [ ] Навбар прозрачный поверх hero → solid при скролле
   - [ ] Карточки объектов: фото + цена + specs + badge
   - [ ] WhatsApp кнопка в правом нижнем углу (зелёная, пульсирует)
   - [ ] Языковой переключатель в навбаре (флаги: 🇷🇺🇬🇧🇪🇸🇵🇹🇨🇳)
   - [ ] Footer с контактами, юр.инфо, соцсетями
   - [ ] Мобильная адаптация (проверить на 375px)

3. **Проверьте бренд-бук:**
   - [ ] Primary: Navy `#1B2838`, Gold `#C9A96E`
   - [ ] Шрифты: Playfair Display (заголовки) + Inter (body)
   - [ ] Отступы: 8px base grid
   - [ ] Border-radius: 8-12px карточки, 4-6px кнопки
   - [ ] Hover: scale(1.02) + shadow, 400ms ease-out

4. **Performance Budget:**
   - [ ] LCP < 1.5s
   - [ ] CLS < 0.05
   - [ ] Lighthouse Score ≥ 95
   - [ ] Total weight < 1.5MB

5. **Сгенерируйте отчёт** с оценкой по каждому критерию (✅/⚠️/❌) и рекомендациями.

## Формат вывода
Таблица с критериями, текущим значением, целевым и статусом.
