# Добавление нового объекта

## Шаги

1. **Подготовка данных**
   - Заполнить все поля по Property Schema (см. skill property-schema)
   - Подготовить минимум 15 фотографий
   - Написать storytelling-описание на EN

2. **Оптимизация медиа**
   - Загрузить фото через Cloudinary MCP
   - Применить auto-format (WebP), auto-quality
   - Установить правильные alt-тексты
   - Упорядочить фото: аэро → экстерьер → интерьер → вид → детали

3. **Перевод контента**
   - Запустить translate-content skill
   - Проверить адаптацию на каждом языке
   - Проверить SEO ключевые слова

4. **Публикация в CMS (Sanity)**
   - Через Sanity MCP создать документ
   - Заполнить все локализованные поля
   - Привязать медиа из Cloudinary
   - Установить координаты на карте

5. **SEO проверка**
   - Запустить seo-audit skill для новой страницы
   - Проверить Schema.org разметку
   - Убедиться что sitemap обновился

6. **Preview и деплой**
   - Запустить deploy-preview skill
   - Проверить карточку на всех 5 языках
   - Мерж в production

7. **Пост-публикация**
   - Проверить в Google Search Console (индексация)
   - Опубликовать в соцсетях (Instagram, Facebook)
   - Отправить уведомление подписчикам (email)
